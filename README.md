# JavaScript Calculator

A clean, responsive calculator built with **Vanilla JavaScript**, **HTML**, and **CSS**.  
Features **dark mode toggle** and **error handling**.

## Features
- Basic arithmetic: `+`, `−`, `×`, `÷`
- Dark/Light mode toggle
- Responsive design
- No frameworks

## Live Demo
Open `index.html` in your browser

## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript
